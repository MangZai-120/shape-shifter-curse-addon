# Installation Guide

## Prerequisites

* **Minecraft** 1.20.1
* **Fabric Loader**
* **Fabric API**
* **Shape Shifter Curse** (the base mod)

## Installation Steps

1. Make sure Fabric Loader and Fabric API are installed
2. Download and install the **Shape Shifter Curse** base mod
3. Download the **Shape Shifter Curse Addon** `.jar` file
4. Drop the `.jar` into your game's `mods` folder
5. Launch the game

!!! warning "Note"
Make sure you download the `.jar` mod file rather than the source code. Get it from the Releases page.

| Channel | Description |
| --- | --- |

| [MCMOD](https://www.mcmod.cn/download/24327.html) | Available, but the channels above are recommended |

!!! tip "Recommendation"
Test builds on Modrinth and CurseForge are mostly there for visibility. If you want the latest test builds, grab them from **GitHub** — those are updated more often.

## Version Recommendation

!!! note "Stay Updated"
Vanilla forms will keep being touched in future updates, so it is recommended to play on the latest version.

## Compatibility

Same as Shape Shifter Curse — other mod compatibility is unknown. This mod runs in **client + server required** mode.

## Version Numbering

Version numbers follow `MAJOR.MINOR.PATCH`:

| Suffix | Meaning |
| --- | --- |
| no suffix | Stable release |
| `+hotfix.X` | Hotfix release |
| `+beta.X` | Public beta |
| `+test` | Internal test build |
