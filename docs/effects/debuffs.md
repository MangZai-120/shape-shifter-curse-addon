# 减益效果

本模组添加了多种减益/伤害状态效果。

---

## 狐火灼烧

**Fox Fire Burn**

SP 使魔的标志性减益效果。

- **效果类型**：伤害 (DoT)
- **来源**：SP 使魔的苍蓝火环、狐火吐息
- **特点**：同时造成 **灼伤和冻伤** 的诡异效果
- **特性**：无法被水扑灭（灵魂火）

---

## 震慑

**Stun**

SP 野猫的控制效果。

- **效果类型**：控制
- **来源**：SP 野猫的震慑冲刺技能
- **效果**：短暂眩晕/无法行动
- **触发条件**：真实隐身期间使用次要技能键冲刺后落地

---

## 霜凝

**Frost Freeze**

SP 雪狐最强力的控场减益。

- **效果类型**：减速 / 增伤 / 禁疗
- **来源**：SP 雪狐的冰霜技能

| 效果 | 数值 |
|------|------|
| 极度迟缓 | 移动速度降低 **35%** |
| 攻击减速 | 攻击速度降低 **40%** |
| 禁疗 | 无法自然恢复生命 |
| 易伤 | 受到的物理和魔法伤害增加 **35%** |

!!! note "免疫"
    SP 眷属狐免疫此效果。

---

## 霜降

**Frost Fall**

SP 雪狐的持续减速效果。

- **效果类型**：减速
- **来源**：SP 雪狐的霜雪护盾

| 效果 | 数值 |
|------|------|
| 减速 | 移动速度降低 **30%** |
| 视觉 | 雪花粒子拖尾 |

!!! note "免疫"
    SP 眷属狐免疫此效果。
