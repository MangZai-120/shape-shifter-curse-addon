## 文件编号：FF-E-013

### 第80次使魔进化测试

**状态：项目终止 — 最高机密**

---

**课题**：第80次使魔进化测试

**实验负责人**：\<机密\>

### Ⅰ. 实验摘要

本次实验旨在研究新生造物是否可以进行额外的改造。

### Ⅱ. 实验材料

1. **13号实验体**：具有赤狐形态中度变化的人类
2. **月尘纯晶**：研究能否再进行诅咒修改
3. **实验型催化剂**：研究能否加快诅咒进程
4. **链接项圈测试型**：研究能否正常控制

### Ⅲ. 实验环境

紫月：变量不可控，但为必须项。

### Ⅳ. 实验步骤

1. 将实验体固定到月光井中心
2. 强制服用实验型催化剂
3. 在变化后为其佩戴链接项圈
4. 令其服用月尘纯晶

### Ⅴ. 实验结果

| 阶段 | 结果 |
|------|------|
| 阶段1 | 实验体在服用催化剂后表现痛苦，身体的变化比正常速度加快，呈现肉眼可见的变化，周围灵能波动加剧 |
| 阶段2 | 实验体开始抽搐，身体已完全变为赤狐样子，但拥有两条尾巴，周围开始冒出低温蓝火 |
| 阶段3 | 对劫掠者和女巫的攻击性几乎为零，但对村民表现出强烈敌意 |
| 阶段4 | 佩戴项圈后实验体完全受到女巫控制，命令其自残也能遵从 |
| 阶段5 | 食用稀释后月尘纯晶后失去控制，形体变大，迸发出的蓝火同时造成灼伤和冻伤（最低温达零下10℃）。暴走约半分钟后倒地死亡，解剖发现储能器官破裂、内部器官灼伤并冻住 |

### Ⅶ. 实验分析

和先前结果类似，实验体在服用月尘纯晶后死亡。大量实验表明**使魔红狐无法再进行深度改造**，其已到达诅咒承担上限。

### Ⅷ. 建议

停止开发使魔的进一步强化研究。每次实验造成的伤害已让劫掠者们损失了多名重要研究员。就算可以成功令其进食月尘纯晶并进化，也极大概率制造出无法控制的狂暴野兽。

## 文件编号：MR-A-007

### 月髓环能量引导与形态突破测试

**状态：封存研究**

---

**课题**：月髓环能量引导与形态突破测试

**实验负责人**：\<机密\>

### Ⅰ. 实验摘要

本次实验旨在探究古代遗物"月髓环"在特定环境下的激活条件，以及其对诅咒宿主形态的干涉能力。

### Ⅱ. 实验材料

1. **73号实验体**：具有永久狼形态变化的人类
2. **月髓环**：自遗迹回收的完好样本
3. 生命体征监测仪 Mk-II
4. 高浓度拘束力场

### Ⅲ. 实验环境

人工模拟紫月环境（模拟度85%），实验后期尝试引入真实紫月光照。

### Ⅳ. 实验步骤

1. 对实验体进行基础身体检查，记录诅咒侵蚀度
2. 将月髓环佩戴于实验体颈部
3. 逐步提升模拟紫月光照强度，临界点引入真实月光
4. 观察实验体形态变化及生命体征

### Ⅴ. 实验结果

| 阶段 | 结果 |
|------|------|
| 阶段1 | 仅模拟光照时，月髓环无明显反应，表面纹路微亮。实验体表示感觉到"冰冷的视线" |
| 阶段2 | 引入真实紫月光照后，月髓环瞬间爆发银辉，强行刺入实验体皮肤与血肉融合 |
| 阶段3 | 实验体哀嚎，原有狼人形态开始崩解，骨骼碎裂重组。生命力以惊人速度流失，被转化为高密度灵能 |
| 阶段4 | 生命力降至临界点时形态质变，体型暴增至1.5倍，毛发呈晶体般质感，打破了形态枷锁 |
| 阶段5 | 实验体失控，撕碎拘束力场。因生命力耗尽突然倒地，数秒内化为灰烬。仅剩失去光泽的月髓环 |

### Ⅶ. 实验分析

月髓环确实拥有打破诅咒形态枷锁的能力，能赋予宿主远超常规的力量。然而，其能量来源并非外部游离灵能，而是**直接抽取使用者的生命本源**。

> 古籍记载"行者之命，亦为之薪柴"并非修辞，而是**事实陈述**。

在未找到替代能源或降低消耗的方法前，任何佩戴行为等同于自杀。

### Ⅷ. 建议

将月髓环封存至高危收容区。建议后续研究方向转向利用外部充能替代生命力消耗，或寻找能承受此等消耗的特殊体质。**禁止任何非授权人员接触。**
