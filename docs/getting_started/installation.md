# 安装指南

## 前置要求

* **Minecraft** 1.20.1
* **Fabric Loader**
* **Fabric API**
* **幻形者诅咒 (Shape Shifter Curse)** 主模组
* 测试 test

## 安装步骤

1. 确保已安装 Fabric Loader 和 Fabric API
2. 下载并安装 **幻形者诅咒** 主模组
3. 下载 **幻形者诅咒扩展包** 的 `.jar` 文件
4. 将 `.jar` 文件放入游戏的 `mods` 文件夹
5. 启动游戏即可

!!! warning "注意"
请确保你下载的是 `.jar` 格式的模组文件，而不是源代码。请到 Releases（发行版本）页面中下载。

## 下载渠道

| 渠道 | 说明 |
| --- | --- |
| [GitHub Releases](https://github.com/MangZai-120/shape-shifter-curse-addon.git) | 更新最频繁，包含 Alpha/Beta 测试版本 |

| [CurseForge](https://www.curseforge.com/minecraft/mc-mods/shape-shifter-curse-addon) | 正式版发布 |

!!! tip "推荐"
Modrinth 和 CurseForge 上的测试版主要为了引流，如果你喜欢体验最新测试版请去 **GitHub** 上下载，GitHub 内发布的测试版本更新更频繁。

## 版本建议

!!! note "保持最新"
因后续会对原版最新形态进行修改，推荐各位玩家保持最新版游玩。

## 兼容性

和幻形者诅咒一样，其余的兼容性问题未知。本模组运行环境为 **客户端需装，服务端需装**。

## 版本号命名规则

本模组的版本号遵循 `大版本.小版本.修订版` 的格式：

| 后缀 | 含义 |
| --- | --- |
| 无后缀 | 正式版 |
| `+hotfix.X` | 紧急修复版 |
| `+beta.X` | 公开测试版 |
| `+test` | 内部测试版 |
