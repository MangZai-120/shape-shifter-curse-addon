# SP 堕落悦灵

SP 堕落悦灵是悦灵形态的暗面进化，归入 **劫掠阵营**，拥有召唤恼鬼大军的能力。

---

## 基本信息

| 属性 | 说明 |
|------|------|
| 基础形态 | 悦灵（堕落变体） |
| 战斗定位 | 恼鬼大军 / 劫掠阵营 |
| 阵营 | 劫掠者阵营 |

---

## 主动技能

### 召唤恼鬼

**主要技能键**

- 召唤 **2 个恼鬼**
- 恼鬼优先攻击 **非白名单** 上离自己最近的玩家
- 范围内无目标时跟随 SP 堕落悦灵
- 恼鬼在场上存在 **25 秒**
- 全部死亡后进入 **20 秒 CD**

### 战场感知

**次要技能键**

- 蓄力 **1 秒** 后触发
- 高亮 **25 格半径** 球内的所有非白名单生物
- **击落** 所有的投掷物 / 飞行物
- 高亮持续 **8 秒**，全体玩家可见
- CD **25 秒**

---

## 被动天赋

### 战斗属性

| 属性 | 数值 |
|------|------|
| 近战伤害 | 固定增加 **4 点** |
| 远程伤害 | 固定降低 **4 点** |
| 血量 | 与 SP 悦灵持平 |
| 速度 | 与 SP 悦灵持平 |

### 恼鬼协同

- 玩家被攻击时，攻击者被标记为恼鬼目标，持续 **10 秒**
- 玩家攻击目标时，目标被标记为恼鬼目标，持续 **10 秒**
- 被标记的目标会被恼鬼 **优先攻击**

### 飞行逻辑

- 取消 SSC 原版的部分飞行逻辑
- 最高向上/下飞 **15 格**
- 需按住 **空格或潜行** 保持飞行状态
- 飞行消耗 **大量饱食度**
- 地下不按潜行/空格会自动向上飞行至地表
- SP 堕落悦灵 **不会受到窒息伤害**

### 阵营特性

- 归类为 **劫掠阵营**
- 劫掠阵营生物不会攻击它
- 它也无法攻击劫掠阵营生物
- 会 **吓跑村民**

### 其他被动

- 细雪上行走
- 免疫 **摔落伤害**
- 走路 **无脚步声**
- 非冰块方块上行走摩擦力增加 **30%**
- 不会自然消耗 **饥饿值**
- 免疫 **火焰伤害**
- 食用 **熟肉**（取代紫水晶）

## 弱点与限制

| 限制 | 说明 |
|------|------|
| 护甲限制 | 强制无法装备护腿和靴子 |
| 护甲可用 | 可以装备头盔和胸甲 |
| 远程弱化 | 远程伤害固定降低 4 点 |
| 阵营限制 | 无法攻击劫掠阵营生物 |
| 紫水晶取消 | 无法分解紫水晶 |

---

## 战斗定位

**恼鬼大军 / 劫掠阵营控制者**

依靠召唤恼鬼进行持续骚扰和伤害，近战伤害增强使其在贴脸战中也有不错的表现。战场感知技能可以击落投掷物并高亮敌人位置，在团战中具有极高的战略价值。但远程能力被大幅削弱。
