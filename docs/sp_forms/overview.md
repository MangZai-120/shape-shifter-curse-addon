# SP 形态概述

**SP 形态（Special Form）** 是本扩展包的核心内容。每个 SP 形态都是原版形态的进化变体，拥有完全重构的主动技能、被动天赋和独特的战斗风格。

---

## 进化方式

SP 形态的进化分为两种途径：

### 诅咒之月进化（月髓十字环）

适用于与诅咒之月有关的形态。玩家需**同时满足**：

1. 处于 **诅咒之月** 时
2. 处于 **永久形态** 下
3. 使用 [月髓十字环](../items/evolution_items.md#月髓十字环)

适用形态：SP使魔、SP雪狐、SP美西螈

### 进化石进化

适用于非诅咒之月形态。使用 [进化石](../items/evolution_items.md#进化石) 长按右键即可。

适用形态：SP野猫

## 设计理念

各个 SP 形态的技能特色使它们的玩法完全不同：

- 有依靠小体型进人群里 **消耗** 的
- 有靠 **隐身** 接近 **刺杀** 的
- 有依赖 **冰霜控场** 的玻璃大炮
- 有凭借 **魔力系统** 持续输出的法师

专为 PVP 进行的平衡使各形态在对战时都有属于自己的优势——有对群专攻，也有对单专攻。不会有完全无用处的形态。
