# SP 野猫 —— 月光魅影

**Wild Cat SP / Moonlight Phantom（月光魅影）**

> "月光没有脚步声。\
> 当你从敌人视线里消失，真正危险的不是隐身本身，而是他们不知道你会从哪里重新出现。"

SP 野猫是以**真实隐身、破隐爆发、短距突进控制**为核心的刺客形态。它不靠正面硬拼，而是通过隐身规避仇恨、切入目标、制造短暂控制，再用必定暴击窗口完成收割。

## 主动技能

### 真实隐身（主技能）

按下主技能键后先进入短暂预备，再获得真正意义上的隐身：

* 预备阶段持续 **1 秒**，期间移动速度降低 **50%**。
* 预备完成后获得 **5 秒真实隐身**。
* 冷却 **12 秒**。
* 装备隐身斗篷时，真实隐身持续时间增加 **2 秒**，但冷却也增加 **2 秒**。
* 隐身期间使用物品、挥手、攻击或挖掘都会破隐。
* 对怪物目标选择生效：怪物不会把真实隐身中的 SP 野猫作为目标，并会在 tick 中脱战。

隐身中再次按主技能可主动解除，获得 **5 秒必定暴击**与 **速度 II 5 秒**，用于把潜入转化为爆发。

### 震慑冲刺（次要技能）

只能在真实隐身期间使用：

* 立刻移除真实隐身。
* 自身获得 **缓慢 III 1 秒**作为出手硬直。
* 向前突进，水平速度 **1.5**、纵向速度 **0.5**，实战约为短距飞扑。
* 1 秒后震慑 **5 格**范围内非白名单目标。
* 震慑持续 **1.5 秒**。
* 冷却 **12 秒**，并同步主技能冷却显示。

### 墨囊致盲

主手持墨囊右键触发：

* 消耗 **1 个墨囊**。
* 致盲 **4.5 格**范围内非白名单目标 **3 秒**。
* 自身获得 **1 秒真实隐身**。
* 冷却 **4 秒**。

## 被动天赋

### 月夜本能

* 夜晚持续刷新速度 I。
* 白天持续刷新缓慢 I。
* 夜晚攻击伤害额外增加 **1 点**。

### 野性食谱

食用生肉时，饱食度与饱和度收益均提升 **100%**。

### 夜视与脱战

SP 野猫拥有稳定夜视能力；真实隐身期间还会阻止怪物锁定自己，非常适合在夜晚或复杂地形中反复切入与撤离。

### 猫科基础能力

SP 野猫继承猫科路线的轻步、摔落免疫、爬墙、吓退苦力怕等能力，也保留腿甲与靴子不适配的体型限制。这些被动让它更适合绕后、攀爬和复杂地形切入。

### 保命猫尾联动

装备保命猫尾且手上没有不死图腾时，可触发一次防死保护：

* 触发后治疗 **6 点生命**。
* 获得吸收 I **15 秒**。
* 获得真实隐身 **5 秒**。
* 同时获得缓慢、抗性提升、虚弱等短暂保护/代价效果，持续 **5 秒**。
* 物品进入 **3 分钟**冷却。

## 弱点与限制

| 限制 | 说明 |
| --- | --- |
| 生命偏低 | 最大生命值降低 **4 点**，不适合长时间正面换血 |
| 白天迟缓 | 白天会持续受到缓慢 I，机动性明显下降 |
| 破隐条件多 | 使用物品、挥手、攻击、挖掘都会打断隐身 |
| 冲刺绑定隐身 | 震慑冲刺只能在真实隐身中发动，不能当普通位移随便交 |
| 护甲限制 | 继承小型猫科体型限制，腿甲与靴子使用受限 |

## 战斗定位

**隐身刺客 / 暗夜切入者**

推荐循环：

1. 开启 **真实隐身**，避开怪物仇恨或玩家视线。
2. 靠近目标后用 **震慑冲刺**制造 1.5 秒控制。
3. 主动解除隐身，触发 **必定暴击 + 速度 II**。
4. 打完爆发后利用地形、墨囊致盲或下一轮隐身撤离。

SP 野猫强在先手与节奏，不强在站桩。只要被迫在白天或开阔地里正面对砍，低生命与破隐限制都会被放大。
