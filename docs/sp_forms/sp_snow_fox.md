# SP 雪狐 —— 霜寒之灵

**Frost Spirit / 凛冬之灵**

> "霜寒之灵——sp雪狐"

掌控极寒的法师，能够用冰霜控制战场。以生命值为代价换取超强的控场和伤害输出能力。

---

## 进化条件

| 条件 | 要求 |
|------|------|
| 基础形态 | 雪狐（永久形态） |
| 进化道具 | [月髓十字环](../items/evolution_items.md#月髓十字环) |
| 天气条件 | 诅咒之月 |

---

## 被动天赋

### 玻璃大炮

- 最大生命值减少 **4 颗心**（-8 HP）
- 以高风险换取高机动性和高伤害

### 霜雪护盾

- 被暴风雪环绕，提供防护
- 自动对靠近的敌人施加减益效果

---

## 减益技能

SP 雪狐拥有强大的 **冰霜控场** 能力，可以对敌人施加两种独特的减益效果：

### 霜凝

最强力的减益效果之一。

| 效果 | 数值 |
|------|------|
| 极度迟缓 | 移动速度降低 **35%** |
| 攻击减速 | 攻击速度降低 **40%** |
| 禁疗 | 无法自然恢复生命 |
| 易伤 | 受到的物理和魔法伤害增加 **35%** |

### 霜降

持续性减速效果。

| 效果 | 数值 |
|------|------|
| 减速 | 移动速度降低 **30%** |
| 视觉 | 雪花粒子拖尾 |

---

## 特殊互动

### SP 眷属狐

SP 雪狐的忠诚同伴，不受严寒侵扰。

- 免疫 **霜凝** 效果
- 免疫 **霜降** 效果

---

## 专属装备

### 雪球发射器

重机枪级别的远程压制武器。

| 属性 | 数值 |
|------|------|
| 射速 | 每 0.35 秒一发 |
| 弹容量 | 20 发 |
| 弹药消耗 | 背包中的雪球 |
| 特点 | 高击退压制 |

### 暴风雪储罐（便携式冰箱）

雪球发射器的完美搭档，无限弹药背包。

| 属性 | 数值 |
|------|------|
| 容量 | 64 发 |
| 自动装填 | 每秒为发射器补充 2 发弹药 |
| 自充能 | 每 2 秒恢复 1 点冰霜储备 |
| 燃料需求 | 无需燃料 |

---

## 战斗定位

**玻璃大炮 / 冰霜控场**

以降低生命值为代价换取极强的控场能力。霜凝效果几乎可以让目标丧失战斗力（减速 + 禁疗 + 易伤），配合雪球发射器的远程压制，在团战中可以有效地控制敌人的行动。但生命值极低，需要保持距离，避免被近身。

---

## 弱点与限制

| 限制 | 说明 |
|------|------|
| 生命值降低 | 最大生命值减少 4 颗心 |
| 脆弱 | 被近身容易被秒杀 |
