# 吸血蝙蝠 —— 嗜血者

> "当月髓环之力浸染了蝙蝠的本质，它便不再仅仅是黑夜的过客……\
> 你成为了嗜血的吸血鬼，月夜的猎手，听不见的低语。"

吸血蝙蝠是蝙蝠在**月髓十字环**之力浸染下蜕变出的特殊变体。它把蝙蝠的夜行本能推向极致：以血渴值驱动的近战循环、可化雾规避伤害的保命位移，以及让对手"听不见声音"的超声波控制。它是一个昼伏夜出、越战越强、但畏惧阳光与铁器的高机动近战形态。

## 主动技能

### 幽雾化形（主技能）

吸血鬼的招牌保命与突进手段：

* 按下主动技能键化为一团幽雾，模型消散无形。
* 雾化期间除**虚空伤害**外**免疫一切伤害**，持续约 **4.5 秒**。
* 化雾 **1 秒**后再次按主动技能键可凝聚爆破，对周围 **4 格**内敌人造成 **8 点伤害**并击退。
* 冷却 **10 秒**。

### 声破超波（次要技能）

向前方释放扇形超声波进行控制：

* 范围 **3.5 格宽、8 格长**。
* 对范围内**非白名单**生物造成 **4 点物理伤害**，并施加 **3 秒反胃**。
* **玩家目标**额外获得 **3 秒失聪**（暂时听不到任何声音）。
* **非玩家目标**额外获得 **3 秒失明**。
* 冷却 **8 秒**。

***

## 被动天赋

### 血渴本能

作为吸血鬼，战斗会让你的血渴值不断累积——普攻命中、技能命中、击杀敌人皆可累加：

* 血渴值低于 **25** 时获得 **15% 伤害减免**。
* 达到 **50** 以上时获得 **20% 吸血**。
* 满 **75** 后伤害提升 **15%**，且吸血提升至 **35%**。
* 脱战 **12 秒**后血渴值开始衰减。

### 月夜亢奋

在夜晚且头顶天空无遮挡时，月光唤起你的猎食本能：攻击力 **+15%**、移动速度 **+5%**，且脱战时血渴值衰减减半。

### 回声定位

夜幕降临后自动启动（不论室内室外）：每 **3 秒**对周围 **12 格**内的所有生物施加一次 **1 秒发光**效果，循环不断，让你在黑暗中追踪每一个目标。

### 继承自蝙蝠的能力

保留普通蝙蝠的几乎所有能力：滑翔与缓慢下落、夜视、墙壁附着、空中加速、跳跃增强、徒手快速挖掘、声呐听声，以及手持镶嵌爪时挖掘钻石的能力。

## 弱点与限制

| 限制 | 说明 |
| --- | --- |
| 日光虚弱 | 白天且头顶天空无遮挡时陷入虚弱：攻击力 **-20%**、移动速度 **-10%**，脱战血渴衰减加快 **50%**；进入阴影、水下或方块掩体后立刻恢复 |
| 银血脆性 | 被持有铁制武器（铁剑/铁斧/铁镐/铁锹/铁锄）的攻击者命中时，受到的伤害额外增加 **15%** |
| 护甲限制 | 蝙蝠的小巧身板，无法装备胸甲、腿甲与靴子，也无法使用远程武器 |
| 地面迟缓 | 在地面行走时移动速度被削弱——天空才是你真正的天地 |

## 战斗定位

**昼伏夜出的高机动吸血近战**

吸血蝙蝠的核心循环是：

1. 趁**夜晚**借月夜亢奋与回声定位进场，用普攻、技能快速堆叠**血渴值**。
2. 血渴越高，伤害与吸血越强，形成"越打越猛"的滚雪球。
3. 受到威胁时用**幽雾化形**免伤脱身或贴脸爆破，必要时用**声破超波**让对手反胃 / 失聪 / 失明打断节奏。

它在夜间是顶尖的近战猎手，但白天和面对铁器对手时格外脆弱——选择交战时机，是吸血蝙蝠的生存关键。
