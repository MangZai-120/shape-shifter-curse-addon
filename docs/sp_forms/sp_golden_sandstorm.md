# 金沙岚

**Golden Sandstorm / 金沙岚**

> "金色的沙并不温暖。\
> 它会附着在伤口里，变成烙印；当你引爆它时，敌人才会明白自己早已被风暴记住。"

金沙岚是阿努比斯之狼三阶段的另一条进化路线。它不依赖死亡领域铺场，而是通过**侵蚀烙印、凋零金沙、百分比爆发、战斗回血**完成近战滚雪球。

## 主动技能

### 凋零金沙（主技能）

蓄力后释放金色沙暴，给敌人挂上第一层侵蚀烙印：

* 蓄力 **1 秒**。
* 蓄力期间移动速度降低 **50%**。
* 影响半径 **15 格**。
* 对非白名单目标施加沙盲 **3 秒**。
* 命中目标叠加 **1 层侵蚀烙印**。
* 成功后冷却 **26 秒**。
* 受伤打断时进入 **7 秒**短冷却。

### 引爆标记（次要技能）

引爆所有非绿色冷却状态的侵蚀烙印目标：

* 没有可引爆目标时不进入冷却。
* 成功后冷却 **10 秒**。
* 每个目标受到当前生命值 **20%** 的伤害，上限 **20 点**，最低 **1 点**。
* 装备枯沙指环时，单次上限提高到 **26 点**。
* 3 层目标会先触发被动爆发，再结算引爆。
* 自身回复 **20% 已损生命值**。
* 装备蚀沙棱晶时，主目标承受 **60%**伤害，剩余 **40%**会扩散给周围目标。

## 被动天赋

### 侵蚀烙印

近战攻击会自动为目标叠加侵蚀烙印：

* 最大 **3 层**。
* 持续 **10 秒**，重复叠加会刷新持续时间。
* 默认同一目标 **1 秒**内最多叠 1 层。
* 装备蚀沙棱晶时，叠层间隔变为 **1.3 秒**。
* 满 3 层后下一次攻击触发被动爆发：造成目标当前生命值 **20%** 的伤害，上限 **20 点**；枯沙指环提高到 **26 点**。
* 被动爆发后治疗自身 **10% 最大生命值**。
* 爆发后目标进入绿色状态 **10 秒**，其中前 **5 秒**不能继续叠层。

蚀沙棱晶会强化爆发扩散：爆发或引爆时把 **40%**伤害扩散给 **5 格**内非白名单目标，并在 **4 格**内额外叠 1 层烙印。

### 烙印感知

被烙印目标会根据状态显示不同颜色：

| 颜色 | 含义 |
| --- | --- |
| 黄色 | 1 层烙印 |
| 橙色 | 2 层烙印 |
| 红色 | 3 层烙印，可触发爆发 |
| 绿色 | 爆发后的叠层冷却状态 |

### 反噬冲击

受到攻击时自动触发反击：

* 影响半径 **4 格**。
* 水平击退 **0.5**，上抛 **0.15**。
* 对非白名单目标施加凋零 I **5 秒**。
* 冷却 **15 秒**；即使没有命中目标也会进入冷却。

### 凋零适应

免疫凋零、失明、生命恢复状态效果与沙盲效果。这里的“生命恢复”指 Regeneration 状态效果；金沙岚自身仍然可以通过专属恢复机制回血。

### 特殊恢复

金沙岚禁用饱食度自然回血，改用专属恢复：

| 来源 | 回复量 |
| --- | --- |
| 自己施加的凋零造成伤害 | 每次回复 **1 点生命** |
| 玩家直接击杀目标 | 回复 **4 点生命** |
| 冥狼击杀目标 | 回复 **3 点生命** |
| 战斗外调息 | 每 **10 秒**回复 **1 点生命** |
| 战斗中调息 | 每 **6 秒**回复 **1 点生命** |

## 弱点与限制

| 限制 | 说明 |
| --- | --- |
| 需要叠层 | 百分比爆发依赖侵蚀烙印，目标频繁脱战会降低输出 |
| 主技能怕打断 | 凋零金沙需要 1 秒蓄力，受伤会转入短冷却 |
| 自然回血被禁用 | 无法依赖饱食度自然回血，必须通过凋零、击杀或调息恢复 |
| 白名单边界 | 近战叠层本身不检查白名单，但扩散、引爆、反噬等范围影响会保护白名单目标 |
| 护甲限制 | 四足身体不适合普通腿甲与靴子 |

## 战斗定位

**烙印爆发型近战**

金沙岚的常用循环是：

1. 用 **凋零金沙**给一片目标挂沙盲与 1 层烙印。
2. 近战持续叠到 3 层，触发 **20% 当前生命值**的被动爆发。
3. 用 **引爆标记**结算所有非绿色烙印目标，并回复已损生命。
4. 被围攻时依靠 **反噬冲击**和凋零回血稳住血线。

它适合中近距离持续压迫，不适合频繁空挥或追逐过于分散的目标。
