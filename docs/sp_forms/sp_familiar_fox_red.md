# 红 —— 只是一个传说

> "红色不是退化，而是另一种回答。\
> 当灵界之主的火焰摆脱既定誓约，它会更锋利、更短促，也更像一场即将失控的自由。"

红是 SP 使魔在被诅咒之月中可能触发的限时稀有分支。它继承 SP 使魔的魔力、火环与收割循环，但体质更脆，狐火吐息与护符火环的伤害更高，整体更像短时间压迫敌人的高风险版本。

***

## 形态定位

| 项目 | 说明 |
| --- | --- |
| 注册名 | `my_addon:form_familiar_fox_red` |
| 分类 | SP 使魔平行分支 |
| 阵营 | 使魔红狐 / 限时红分支 |
| 战斗定位 | 高风险持续 AOE / 强化狐火吐息 |

## 进化条件

红有两条获取路径：**限时随机分支**与**月痕之力剧情路径（永久）**。

### 路径一：限时随机分支

| 条件 | 要求 |
| --- | --- |
| 基础形态 | SP 使魔（`my_addon:form_familiar_fox_sp`） |
| 触发环境 | 被诅咒之月期间 |
| 分支判定 | 每次被诅咒之月期间首次检查有 **5%**几率转为红 |
| 持续时间 | 转化后获得 `ssc_addon_red_expire` 标记，持续 **12000 tick（10 分钟）**，到期回到 SP 使魔 |
| 备注 | 月髓十字环升级代码中也有红分支判定，但随后仍会继续执行普通 SP 转化；本路径按稳定生效的 SP 使魔咒月转化逻辑说明 |

### 路径二：月痕之力剧情路径（永久）

| 条件 | 要求 |
| --- | --- |
| 前置形态 | SP 使魔（`my_addon:form_familiar_fox_sp`） |
| 前置成就 | 解锁隐藏成就「**月痕之力**」——集齐全部与灵界之主相关的书籍：《月痕：灵界之主》第一章~尾声共 9 本 + 《使魔研究记录 FF-E-013》 |
| 引导提示 | 解锁成就且处于 SP 使魔时，会收到一次低语提示，引导你去睡觉 |
| 触发方式 | 在**诅咒之月的夜晚睡觉**：SP 使魔会真正进入睡眠（沿用四足形态的睡觉动画），睡醒后转化为红 |
| 持续时间 | **永久**——不会像限时分支那样到期回退 |
| 变回方式 | 处于剧情红状态时，可随时随地使用**月髓十字环**免费变回 SP 使魔（不消耗物品） |
| 备注 | 此路径不打 `ssc_addon_red_expire` 标记，因此不会被限时分支的到期逻辑回退；两条路径相互独立 |

## 主动技能

### 苍蓝火环（主技能）

红的普通火环与 SP 使魔接近，但狐火持续时间不同：

* 启动需要魔力达到 **99 点**，关闭、魔力不足或被净化打断后进入 **20 秒**冷却。
* 影响自身周围 **6 格**范围。
* 每 **4 tick**进行一次维持判定，魔力不低于 **16 点**时消耗 **1.7 魔力**，不足则自动关闭并进入冷却。
* 伤害结算约每 **16 tick**一次，对非白名单目标造成 **4 点伤害**，并施加 **4 秒狐火灼烧**。
* 火环开启期间击杀目标会额外回复 **5 魔力**并治疗 **3 点生命**。

装备蓝火护符时改用红的护符强化版火环：启动需要 **119 魔力**，每 **4 tick**消耗 **2.8 魔力**，半径 **3.6 格**，约每 **16 tick**结算一次 **7 点伤害**并施加 **3 秒狐火灼烧**，关闭或中断后冷却 **28 秒**。

### 赤焰狐火吐息（次要技能）

红的吐息比普通 SP 使魔更凶狠：

* 消耗 **20 魔力**。
* 射程 **8 格**。
* 命中造成 **7 点伤害**。
* 附加 **6 秒**狐火灼烧。
* 冷却 **1 秒**，并同步显示在 SP 次要技能 CD 条上。

## 被动天赋

### 灵魂收割

* 击杀目标基础回复 **10 魔力**。
* 苍蓝火环开启时，击杀额外回复 **5 魔力**并治疗 **3 点生命**。

### 灵能护幕

红继承 SP 使魔的魔力减伤：魔力不低于 **10 点**时获得 **35% 伤害减免**；高魔力状态下额外获得 **20% 伤害减免**。

### 灵体化身

继承高阶灵体的异常状态抗性与使魔系基础能力，但这并不能完全抵消它更薄的生命池。

### 药水袋

处于红形态时，第 9 个快捷栏槽位会自动获得药水袋；离开红形态后会移除药水袋，并将袋内物品掉落出来，避免形态专属物品被长期保留。

### 脆弱体质

红额外拥有最大生命值 **-4** 的调整。它能打出更高的吐息爆发，但容错比 SP 使魔更低。

***

## 弱点与限制

| 限制 | 说明 |
| --- | --- |
| 生命更低 | 最大生命值额外降低 **4 点**，被爆发伤害命中时更危险 |
| 魔力依赖 | 火环与吐息都消耗魔力，低魔会触发与 SP 使魔相同的失衡惩罚 |
| 进场风险 | 火环需要靠近目标，红的生命惩罚会放大贴身失误 |
| 护甲限制 | 继承使魔体型限制，无法正常使用腿甲与靴子 |
| 水域行动 | 不适合长时间水下作战 |

***

## 战斗定位

| 维度 | SP 使魔 | 红 |
| --- | --- | --- |
| 获得方式 | 月髓十字环进化 | SP 使魔在被诅咒之月中 5% 限时转化 |
| 普通火环 | 6 格，约每 16 tick 造成 4 伤害，狐火 3 秒 | 6 格，约每 16 tick 造成 4 伤害，狐火 4 秒 |
| 护符火环 | 3.6 格，约每 20 tick 造成 6 伤害 | 3.6 格，约每 16 tick 造成 7 伤害 |
| 吐息 | 8 格，6 伤害 | 8 格，7 伤害，狐火 6 秒 |
| 生存 | 无额外生命惩罚 | 最大生命值额外 -4 |

红的核心打法与 SP 使魔接近，但更强调用强化吐息压低敌人血线，再靠苍蓝火环与灵魂收割滚动续航。它不适合拖到低魔，也不适合吃下正面爆发；进场时机比普通 SP 使魔更重要。
