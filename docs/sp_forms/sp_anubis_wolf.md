# SP 阿努比斯 —— 冥裁者

**SP Anubis / Netherjudge（冥裁者）**

> "你不再只是亡灵狼群的首领。\
> 当灵魂沙在脚下铺开，审判就有了边界；踏入边界的灵魂，都要付出代价。"

SP 阿努比斯是以**死亡领域、冥狼召唤、灵魂能量强化**为核心的区域控场形态。它能把战场暂时改造为灵魂沙领域，并借此削弱敌人、增强冥狼、扩大亡灵系压制力。

## 主动技能

### 死亡领域（主技能）

蓄力后把周围环境暂时改造为亡灵主场：

* 普通蓄力 **2 秒**，蓄力期间移动速度降低 **70%**。
* 普通半径 **24 格**，高度范围 **9 格**。
* 领域以约 **5 格/秒**速度向外扩张。
* 普通持续 **15 秒**。
* 成功后冷却 **50 秒**。
* 若没有可转化方块导致释放失败，进入 **10 秒**惩罚冷却。
* 方块转换会避开基岩、功能方块、红石与方向连接类方块，领域结束后还原。

领域中，站在灵魂沙或灵魂土上方约 3 格范围内的非白名单目标会受到：

* 缓慢 I 刷新。
* 凋零 I；强化领域中提升为凋零 II。
* 最大生命值降低 **15%**。
* 凋零打击累计上限约 **10 次**，避免无限滚伤害。

满灵魂能量时释放会变为强化死亡领域：

* 消耗全部灵魂能量。
* 蓄力缩短为 **1 秒**。
* 半径扩大到 **32 格**。
* 持续 **25 秒**。
* 自动召唤 **6 只冥狼**；拥有阿努比斯水晶时额外 +2 只。
* 同时让冥狼裁庭进入 **30 秒**冷却。

### 冥狼裁庭（次要技能）

召唤冥狼为自己作战：

* 蓄力 **1.5 秒**，蓄力期间移动速度降低 **50%**。
* 基础召唤 **2 只**冥狼。
* 死亡领域中召唤 **4 只**。
* 冥狼存活 **30 秒**。
* 成功后冷却 **30 秒**。
* 达到召唤上限时释放失败，进入 **5 秒**短冷却。
* 默认最多同时存在 **6 只**；拥有阿努比斯水晶时上限 +2。

冥狼基础属性为生命 **20**、攻击 **4**，攻击命中会治疗主人 **2 点生命**。在死亡领域中召唤的冥狼额外获得 **+10 生命**、**+2 攻击**与速度 I。阿努比斯水晶会提高数量上限与召唤数量，但冥狼攻击降低 **25%**、生命降低 **35%**。

## 被动天赋

### 灵魂能量

灵魂能量范围为 **0-100**。满能量时会强化下一次死亡领域，把普通控场升级为大范围领域与冥狼爆发窗口。

### 灵魂审视

**16 格**内生命值不高于 **20%** 的生物会发光，并周期性出现灵魂粒子提示。

### 死亡宣告

攻击相对生命值不高于 **30%** 的目标时，伤害提高 **20%**。

### 凋零追猎

当 **24 格**内存在带凋零效果的生物时，在非灵魂沙/灵魂土区域获得 **20% 移动速度**加成。

### 亡灵体魄

最大生命值增加 **4 点**，并继承阿努比斯之狼路线的亡灵本能与灵魂沙相关能力。

## 弱点与限制

| 限制 | 说明 |
| --- | --- |
| 强依赖领域 | 主要压制力来自死亡领域，技能空掉或被迫离场会明显降强度 |
| 蓄力很重 | 普通领域蓄力 2 秒且减速 70%，容易被远程或爆发打断节奏 |
| 方块保护 | 无可转化方块时领域会失败并进入短冷却 |
| 召唤上限 | 冥狼数量受上限限制，不能无限堆叠 |
| 护甲限制 | 四足身体不适合普通腿甲与靴子 |

## 战斗定位

**亡灵领域召唤师**

SP 阿努比斯最强的窗口来自强化死亡领域：更大的领域、更长持续时间、凋零 II 与自动冥狼一起压上去。普通战斗中，它也可以用死亡领域限制走位，再用冥狼裁庭补伤害和吸血。适合据点战、Boss 战和需要限制敌人移动的多人战场。
