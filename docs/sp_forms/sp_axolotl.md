# SP 美西螈 —— 深渊霸主

**Axolotl SP / Abyssal Overlord（深渊霸主）**

> "深渊不是安静的水。\
> 它会把敌人拖到你身边，再用一次爆破告诉他们，什么才叫水下的王权。"

SP 美西螈是围绕**湿润值、生存减伤、拉拽爆发**设计的水系形态。它能用涡流把敌人卷入近身范围，也能用假死中断怪物仇恨，在多人战斗中承担控场与救命缓冲的职责。

## 主动技能

### 涡流冲击（主技能）

以自身为中心制造涡流，先拉拽再爆破：

* 冷却 **20 秒**。
* 消耗 **20% 湿润值**。
* 涡流持续 **3 秒**。
* 持续期间拉拽 **6 格**范围内非白名单目标。
* 技能期间自身受到的伤害降低 **50%**，但移动速度降低 **50%**。
* 结束时在 **3 格**范围内爆发，造成 **12 点伤害**。
* 爆发击飞参数已降低为当前代码值：纵向 **0.6**，水平冲击 **0.8**。
* 命中后施加 **缓慢 III 4 秒**。

该技能使用默认白名单机制，白名单保护目标不会被拉拽、伤害或控制。

### 假死（次要技能）

主动进入 6 秒假死状态，为自己争取脱战窗口：

* 冷却 **36 秒**。
* 持续 **6 秒**。
* 期间获得 **90% 伤害减免**。
* 禁止跳跃、攻击、挖掘和使用物品。
* 自身获得生命恢复 III、失明与极高等级缓慢。
* 清除 **64 格**内以自己为目标的怪物仇恨。
* 装备活性珊瑚项链时，假死期间的生命恢复改为 I，并额外获得吸收 V **25 秒**。

### 水矛制作

副手持箭时右键，可以把箭转化为水矛：

* 消耗 **10% 湿润值**与 **1 支箭**。
* 背包内没有水矛时才会制作。
* 冷却 **5 秒**。

### 强化水流爆破

在地面上从奔跑切换为潜行时触发水流爆破：

* 爆炸强度 **3**。
* 消耗 **5% 湿润值**。
* 使用默认白名单机制，避免误伤受保护目标。

## 被动天赋

### 雨润

雨中每 **20 tick** 回复 **60 air**，可以显著缓解陆地作战的湿润压力。

### 快速代谢

每 **20 tick** 自然治疗 **0.25 点生命**，为长时间消耗战提供稳定恢复。

### 湿润值循环

SP 美西螈的主技能、爆破与水矛都消耗湿润值。离水作战时需要利用雨天、加湿器或水域环境维持资源，否则会被迫停止主动技能循环。

### 美西螈基础能力

SP 美西螈继承美西螈路线的水下呼吸、湿滑/游泳姿态、水中机动、跃出水面、氧气生命与食性相关限制。它在水中更完整，离水后则需要持续管理湿润值与行动节奏。

## 弱点与限制

| 限制 | 说明 |
| --- | --- |
| 湿润依赖 | 涡流、水矛与水流爆破都会消耗湿润值，陆地长战需要补水手段 |
| 涡流站位 | 涡流以自身为中心，想打满控制就必须靠近敌群 |
| 假死锁定动作 | 假死期间无法攻击、挖掘、跳跃或使用物品，只适合救急与脱战 |
| 爆破触发条件 | 强化水流爆破依赖地面奔跑转潜行，复杂地形中需要熟悉节奏 |
| 装备限制 | 继承美西螈路线限制，头盔与部分常规装备不适配 |

## 战斗定位

**深渊控场坦克**

SP 美西螈的核心循环是先用涡流把敌人卷入身边，再靠爆破伤害与缓慢 III 创造队友输出窗口。假死提供强力保命，水矛和水流爆破则补足中短距离手段。它最怕资源干涸，所以陆地作战要提前准备补水工具。
