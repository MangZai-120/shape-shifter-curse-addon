# SP 美西螈 —— 深渊霸主

**Deep Sea Lurker / 深海潜伏者**

> "深渊霸主——sp美西螈"

专注于生存和水下掌控的特化进化。在水中，你就是无可争议的王者。

---

## 进化条件

| 条件 | 要求 |
|------|------|
| 基础形态 | 美西螈（永久形态） |
| 进化道具 | [月髓十字环](../items/evolution_items.md#月髓十字环) |
| 天气条件 | 诅咒之月 |

---

## 主动技能

### 假死

主动进入"假死"状态。

- 所有怪物仇恨立即消失
- 伴随 **失明** 效果
- 获得 **90% 减伤**
- **快速恢复** 生命值

### 强化水流爆破

主动水流攻击。

- 伤害远高于标准美西螈形态

### 涡流冲击

强力的 AOE 主动技能。

- **吸附** 周围敌人
- 造成 **高额伤害**
- 释放期间自己 **减速 50%**
- 获得 **30% 伤害免疫**

### 水矛

远程武器技能。

- **获取方式**：副手持箭右键
- **持续时间**：1 分钟
- **效果**：命中造成伤害并附带减速
- **溅射**：小范围内敌人也受到额外伤害

---

## 被动天赋

### 雨润

直接暴露在雨中可以恢复湿润度，使风暴期间的陆地旅行成为可能。

## 专属物品

### 美西螈加湿器

SP 美西螈形态的救命道具。在离水作战时维持你的湿润度。

| 属性 | 说明 |
|------|------|
| 合成材料 | 铁锭 + 红石 + 海绵 |
| 使用方式 | 手持右键切换开关 |
| 充能方式 | 使用美西螈桶合成补充 |
| 持续时间 | 最长 90 分钟 |

---

## 战斗定位

**水下霸主 / 生存专家**

在水中拥有无与伦比的战斗力，假死技能提供强大的生存能力。涡流冲击适合群战控场，水矛提供远程输出。通过加湿器在陆地上也能保持战斗力。
