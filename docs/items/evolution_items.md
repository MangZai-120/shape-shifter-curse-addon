# 进化用品

进化用品是 SP 形态进化的关键道具。不同的形态需要不同的进化用品来触发进化。

---

## 月髓十字环

**Moon Essence Cross**

| 属性 | 值 |
|------|-----|
| 物品命令 | `/give @p ssc_addon:sp_upgrade_thing 1` |
| 最大堆叠 | 1个/组 |
| 分类 | 进化用品 |

### 描述

> "一枚凝结着月华与诅咒的金色十字。当紫色的月光笼罩大地，唯有挣脱血肉束缚的幻形者，方能在月相交汇的刹那触碰其真正的力量——或是蜕变为超越凡躯的形态，或是被月髓中奔涌的狂躁能量撕裂魂魄，炸裂成一场璀璨而致命的星尘雨。此物非钥非刃，它是与月华签订的双生契约：一半是升华的辉光，一半是湮灭的冰裂。——谨记，唯有月光为尊。"

### 合成配方

使用工作台合成：

| 材料 | 数量 |
|------|------|
| 金锭 | 4 |
| 绿宝石 | 1 |
| 红石粉 | 1 |
| 塑形核心 | 1 |
| 月尘纯晶 | 1 |
| 下界合金锭 | 1 |

!!! note "布局"
    除金锭和塑形核心外的材料可无序合成。金锭放在四角，塑形核心放在中间。

### 使用条件与效果

此物品用于**诅咒之月相关形态**的进化。玩家需**同时**满足以下条件才能成功使用：

| 条件组合 | 效果 |
|----------|------|
| 两种都不满足 | 无效果，月髓环消失 |
| 只满足诅咒之月 | 无效果，月髓环消失 |
| 只满足永久形态 | 无效果，月髓环消失 |
| **两者都满足** | **进化为 SP 形态，月髓环消失** |
| 拥有SP形态 + 不满足诅咒之月 | 无效果，月髓环消失 |
| 拥有SP形态 + 满足诅咒之月 | 方块破坏性爆炸（和平模式无伤害，其他难度造成100000点伤害致死） |

!!! warning "危险"
    月髓环的爆炸效果和末影水晶类似。爆炸时对使用它的玩家造成 **100000 点伤害**。可以使用不死图腾来免疫月髓环爆炸的死亡，所以不乏为一种应急的输出手段。

### 适用形态

- SP使魔（灵界之主）
- SP雪狐（霜寒之灵）
- SP美西螈（深渊霸主）

## 进化石

**Evolution Stone**

| 属性 | 值 |
|------|-----|
| 物品命令 | `/give @p ssc_addon:evolution_stone 1` |
| 最大堆叠 | 1个/组 |
| 分类 | 进化用品 |

### 描述

> "这是一颗仿佛自最深梦境中打捞上来的宝石，其形态宛若一颗永恒的、凝望深渊的猫瞳。在静谧的暗夜中，你能窥见其内部瑰丽的宇宙：幽匿的青色脉络如命运的神经般蔓延，猎杀的猩红余烬在核心缓缓跳动，夜视的流萤微光与终幕的虚空星云在其间交织盘旋，而一切光芒最终都被暗影的无垠基石温柔吞噬，归于一片沉静的漆黑。"

> "当五种历经磨难的暗影碎片在仪式中合而为一，它们便不再是凡俗的造物。它拥有脉搏与呼吸，仿佛一颗沉睡的心脏，在掌心低语着关于潜行、猎杀、坚韧、视野与虚空的古老歌谣。它是暗夜权能的终极结晶，只为等待一位能与之共鸣的宿主，完成那场命中注定的觉醒。"

### 合成配方

使用工作台合成，由五种 [进化碎片](materials.md) 合成：

| 材料 | 数量 |
|------|------|
| 猎杀碎片 | 1 |
| 幽匿碎片 | 1 |
| 终幕碎片 | 1 |
| 夜视碎片 | 1 |
| 暗影碎片 | 1 |

### 使用方式

长按右键使用。用于**非诅咒之月形态**的进化。

- 若是**非指定形态**使用：提示失败，无事发生
- 若是**指定形态**使用：提示成功，消耗进化石

### 适用形态

- SP野猫（暗夜猎手）
